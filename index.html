Help Guide

What Is This?

The CieloVista Website Generator creates complete, production-quality single-file HTML websites from a plain-English subject. Type a topic, configure a few options, and within 2–3 minutes you have a fully designed, self-contained .html file.

Requirements

Set these as Windows environment variables (Start → "environment variables" → User variables → New), then restart the server:

VariablePurposeRequired?
CLAUDEAnthropic Claude — writes all contentYes
falaifal.ai FLUX — generates imagesOptional
OPENAIOpenAI — fallback content providerOptional

Images

Requires the falai environment variable. When enabled, Claude writes grounded image prompts after generating content — a hero image plus one per card. All images embed as base64 in the HTML so the file is fully self-contained.

My Sites — Restyle

Each gallery card has a ⚙ Restyle button. Expand it to pick a new layout, toggle sections, and change the images setting, then click ↺ Regenerate to rebuild the site with Claude auto-selecting the visual style.

Tips

  • Be specific with your subject — more detail = better content.
  • Use the Context field to steer the angle Claude takes.
  • Include Timeline for historical subjects — it anchors accuracy.
  • Style is always Auto — Claude picks the best fit for your subject.

Troubleshooting

Generation fails: Server logs at generated/_shared/artifacts/logs/ai-trace.log. Most timeouts are API rate limits — wait a minute and retry.

Images not appearing: Check falai env var is set and the server was restarted after setting it.

Generic content: Add more detail to the Context field, or try a more specific subject.
