# createWebsite: Deterministic Website Generator

This folder will contain a deterministic website generator that allows users to create spectacular websites based on any subject, following the principles of accuracy, reproducibility, and transparency outlined in the previous documents.

## Principles Applied
- Deterministic output: Same input and settings always yield the same website.
- Full logging: All user inputs, settings, and generated files are logged for reproducibility.
- User control: Users can select deterministic or creative modes.
- Transparency: All steps and decisions are explained to the user.

## Structure
- /src — Source code for the generator
- /logs — Logs of all sessions and generated sites
- /templates — Website templates for various subjects
- /README.md — Documentation and usage instructions

---

Start by adding your requirements or subject in the generator UI. The system will guide you through deterministic or creative generation modes, and log all steps for reproducibility.
